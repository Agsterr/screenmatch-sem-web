# ScreenMatch (sem Web)

Versão em linha de comando do projeto **ScreenMatch** da Alura — catálogo de filmes e séries sem interface web.

## O que faz

- Carrega dados de filmes a partir de arquivo JSON
- Permite buscar e filtrar títulos pelo console
- Calcula avaliações e exibe informações dos filmes
- Projeto introdutório de Java com orientação a objetos

## Tecnologias

- Java 17+
- Maven
- Gson (JSON)

## Como rodar

```bash
git clone https://github.com/Agsterr/screenmatch-sem-web.git
cd screenmatch-sem-web
mvn compile exec:java -Dexec.mainClass="com.screenmatch.Principal"
```
